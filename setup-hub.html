n8n Integration
setup

Connect n8n workflows to OpenClaw via webhooks and the OpenAI-compatible API.

Integration endpoints:
# n8n HTTP Request node -> OpenClaw
POST http://localhost:18789/v1/chat/completions
Authorization: Bearer <gateway-token>

# OpenClaw webhook -> n8n
POST http://localhost:5678/webhook/openclaw

# n8n trigger -> OpenClaw agent
POST http://localhost:18789/hooks/agent
x-openclaw-token: <gateway-token>
